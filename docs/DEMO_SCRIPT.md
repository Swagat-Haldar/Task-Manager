# Demo Walkthrough Script

This script outlines a professional 3-5 minute demonstration of the Team Task Manager.

## Scene 1: The First Impression (30s)
- **Action**: Land on the `LoginPage`. 
- **Narration**: "Welcome to Team Task Manager. We've built a premium, workspace-focused platform for high-performing teams."
- **Action**: Log in as `admin@example.com`.
- **Narration**: "Logging in as an Admin, we're greeted by a sleek, modern dashboard showing our team's velocity and overdue tasks."

## Scene 2: Project Management (60s)
- **Action**: Navigate to `Projects`. Show the workspace cards.
- **Action**: Create a new project: "Project Alpha". 
- **Narration**: "Creating a project is instant. As the creator, I'm automatically the Project Admin."
- **Action**: Enter "Project Alpha". 
- **Narration**: "The project view is clean, focusing on objectives and team composition."

## Scene 3: Collaboration & RBAC (90s)
- **Action**: Invite `member@example.com` to the project.
- **Action**: Create a task: "API Integration" and assign it to the member.
- **Narration**: "We can easily manage our team and delegate work with priority levels."
- **Action**: Logout and login as `member@example.com`.
- **Narration**: "Now, switching to a Member view. Notice how the administrative buttons disappear. I have exactly the permissions I need to do my job."
- **Action**: Open "Project Alpha" and update the task status to "In Progress".
- **Narration**: "As a member, I can update my own work, keeping the team synced without the risk of accidentally changing project settings."

## Scene 4: Conclusion (30s)
- **Action**: Return to the Dashboard and show the updated progress bars.
- **Narration**: "The dashboard reflects real-time progress. High-fidelity analytics, robust security, and a beautiful UI. That's Team Task Manager."
